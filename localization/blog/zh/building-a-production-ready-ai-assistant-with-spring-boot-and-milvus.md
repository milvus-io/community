<p>每家公司都面临同样的问题：宝贵的知识被困在 PDF、Word 文档和文件共享中，需要时却找不到。支持团队重复回答同样的问题，而开发人员则浪费大量时间搜索过时的文档。</p>
<p><strong>如果您的文档可以直接回答问题，那会怎样？</strong></p>
<p>本教程将向您展示如何构建一个可用于生产的人工智能助手，它可以</p>
<ul>
<li><p>将静态文档转化为智能问答系统</p></li>
<li><p>保持对话上下文和记忆</p></li>
<li><p>可扩展以处理企业级工作负载</p></li>
<li><p>开箱即包含安全性、监控和可观察性</p></li>
</ul>

<li><p><a href="https://github.com/spring-projects/spring-boot"><strong>Spring Boot</strong></a>是一种广泛使用的 Java 框架，用于以最少的配置构建后端应用程序。它为开发人员提供了强大的生产力、与现代工具的无缝集成，以及对 REST API、可观察性和安全性的内置支持。</p></li>

<li><p><a href="https://ollama.com/"><strong>Ollama</strong></a>：本地人工智能模型提供商（兼容 OpenAI，完全免费）</p></li>
</ul>
