</ul>

<li><p><a href="https://github.com/spring-projects/spring-boot"><strong>Spring Boot</strong></a>是一個被廣泛使用的 Java 框架，用來以最少的設定建立後端應用程式。它提供強大的開發人員生產力、與現代工具的無縫整合，以及內建的 REST API、可觀測性和安全性支援。</p></li>

<li><p><a href="https://zilliz.com/learn/Retrieval-Augmented-Generation"><strong>RAG</strong></a>是一個結合檢索與產生的架構：它會從 Milvus 等向量資料庫擷取相關的知識片段，然後使用語言模型來製作流暢且符合上下文的回應。</p></li>
<li><p><a href="https://ollama.com/"><strong>Ollama</strong></a>: 本地 AI 模型提供者 (相容於 OpenAI，完全免費)</p></li>
</ul>
